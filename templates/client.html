API Quick Integration Reference

Check VPS Status via cURL

curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  http://{{ request.host }}/api/v1/vps

Trigger VPS Action

curl -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"action": "restart"}' \
  http://{{ request.host }}/api/v1/vps/<vps_id>/action
